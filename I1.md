1 Project 1: Smart Menu & Digital Display System

Executive Summary

(cid:17) A real-time menu management system enabling hospitality venues to instantly update
digital menus and displays via a simple web dashboard, eliminating printing costs and
manual menu updates.

1.1 Problem Statement

Challenge: Small hospitality businesses struggle with dynamic menu management, relying on
expensive printed menus or static PDFs that cannot be quickly updated.

Real-World Scenario: The Anchor Pub

1. Manager discovers "Guest Ale" has run out at 11 AM

2. Logs into admin dashboard at anchorpub.com/admin

3. Updates "Guest Ales" section with new item

4. Instant synchronization:

• QR code menus on tables reflect changes immediately

• TV display above bar auto-refreshes with new information
